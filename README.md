#  代码笔记本

#### 记录常用代码片段，养成你自己的vscode

## usage

- command + shift + p 打开命令面板
- 输入记录的代码片段，即可完成复制，在使用的地方粘贴即可

目前内置命令

- 新增code，添加属于你的代码片段。(注：添加后需要重启vscode才能生效)
- debounce(防抖)
- throttle(节流)
